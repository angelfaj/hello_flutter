# Flutter – Cuaderno de Prácticas

Este repositorio se utiliza como cuaderno de prácticas para un curso de **Flutter** que estoy siguiendo.

## Objetivo

- Practicar los conceptos vistos en el curso
- Probar ejemplos y ejercicios de Flutter y Dart
- Servir como referencia personal de aprendizaje

## Estructura

El repositorio puede contener:
- Ejercicios independientes
- Proyectos pequeños de prueba
- Notas o ejemplos de código

La organización puede cambiar a medida que avance el curso.

## Requisitos

- Flutter SDK
- Dart
- Un entorno de desarrollo compatible (VS Code, Android Studio, etc.)

## Notas

Este repositorio **no pretende ser código de producción**, sino material de práctica y aprendizaje.
